### Triggers a GitHub webhook

## Required Environment Variables

| Key             | Description                                               |
| ----------------|-----------------------------------------------------------|
| `APP_NAME`      | developer or App Name                                     |
| `REPO`          | repository name                                           |
| `OWNER`         | repository owner                                          |
| `EVENT_TYPE`    | webhook event name                                        |
| `TOKEN`         | jwt token to do api calls                                 |
